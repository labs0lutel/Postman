Требования к API
Необходимо реализовать следующие эндпоинты:

Создание задачи
Метод: POST /tasksПринимаемые параметры:title (строка, обязательно)description (строка, опционально)completed (булево, опционально, по умолчанию false)title (строка, обязательно)description (строка, опционально)completed (булево, опционально, по умолчанию false)

Получение списка задач
Метод: GET /tasksПараметры запроса:completed (булево, опционально) - фильтрация по статусуcompleted (булево, опционально) - фильтрация по статусу

Получение конкретной задачи
Метод: GET /tasks/{id}

Обновление задачи
Метод: PUT /tasks/{id}Принимаемые параметры (все опционально):titledescriptioncompletedtitledescriptioncompleted

Удаление задачи
Метод: DELETE /tasks/{id}

Тестирование API

Создание задачи

Получение конкретной задачи
Отправить GET запрос с существующим idОтправить GET запрос с несуществующим id (ожидать 404)

Обновление задачи
Создать задачуОбновить её параметрыПроверить корректность обновления

Удаление задачи
Создать задачуУдалить еёПроверить удаление
